Sistem Pengurusan Data ICT Sekolah

Mudah, pantas, dan mesra pengguna.

Maklumat Sistem

Sistem ini membolehkan pengguna (Guru ICT/Admin sekolah) merekod peralatan ICT dan memantau status dengan mudah dan cepat.

Juga direka khusus untuk kegunaan Pasukan Teknikal ICT Sekolah (PTIS), PPD Kunak bagi meningkatkan kecekapan pengurusan data ICT di sekolah-sekolah.

Sistem akan sentiasa dikemaskini dari semasa ke semasa.

Selamat menggunakan sistem ini!

Notis Penting

  • Pastikan kemaskini data dilakukan 3 bulan sekali (4 Fasa).
  • Kemaskini data segera jika ada perubahan untuk mengelakkan data anda tercicir.
  • Pastikan semua peralatan direkod dengan betul.
  • Hubungi admin jika terdapat ralat data atau masalah mengenai sistem ini.
